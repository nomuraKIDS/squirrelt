# squirrelt

squirrelt は、`ls` コマンドをより使いやすくしたファイル一覧表示ツールです。ファイルの拡張子ごとに色を変えて、目的に応じた並べ替えや隠しファイルの表示を簡単に切り替えられます。

## 主な特徴

- 拡張子ごとに色分けして表示
- ファイルサイズ順、更新日時順、名前順でソート
- `-a` オプションで隠しファイルも表示
- カレントディレクトリや指定したパスの内容を一覧表示

## 使い方

```bash
squirrelt [OPTIONS] [PATH]
```

`PATH` を指定しない場合は、現在のディレクトリを表示します。

## オプション

- `--sort-size` : ファイルサイズの順に並べ替えて表示します。
- `-t` : ファイルの更新日時が新しい順に表示します。
- `-a` : 隠しファイルも含めて表示します。

## こんな人におすすめ

- `ls` の表示をもっと見やすくしたい人
- ファイル一覧をサイズや更新日で並べ替えたい人
- 隠しファイルを手早く確認したい人

## 例

```bash
squirrelt --sort-size
squirrelt -t /path/to/directory
squirrelt -a
```
